# Exercises

These are “small but real” tasks you can do to learn React Native.

## Beginner

1. Add a “pull to refresh” on the Home feed.
2. Add a “bookmark” state to posts.
3. Add a “double tap to like” gesture on the post image.

## Intermediate

1. Add a “profile header” edit button that opens a modal.
2. Add pagination (load more posts when you scroll).
3. Persist posts locally using Expo SQLite or AsyncStorage.

## Advanced

1. Replace mock data with a real backend.
2. Add image picking for Create Post.
3. Add video playback for Reels.
